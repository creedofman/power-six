# Usage

### Online

Go to [powersix.ga](https://powersix.ga) using the **Experimental Browser**

### Offline

-  Clone down the repo.

-  Move everything to a folder named `power-six` inside your kindle drive.

-  Go to your browser and type `file:///mnt/us/power-six/index.html`.
